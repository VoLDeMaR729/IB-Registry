# Реестр компаний (IB Registry)

Система учета компаний в сфере информационной безопасности.
Разработано в рамках курсового проекта.

## Технологический стек
* **Язык:** C++17
* **GUI:** Qt 6 (Widgets)
* **База данных:** PostgreSQL 15 (в Docker)
* **Сборка:** CMake
* **Безопасность:** Хеширование паролей (SHA-256), защита от SQL-инъекций.

## Установка и запуск

### Предварительные требования
* Linux (Ubuntu 22.04+)
* Docker & Docker Compose
* Qt6, CMake, GCC

### Быстрый запуск
В корне проекта есть скрипт автоматического запуска:
```bash
chmod +x run.sh
./run.sh
